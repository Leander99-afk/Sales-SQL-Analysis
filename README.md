# Sales \& Customer Analysis using SQL

## Project Objective

Analyze sales transaction data to identify revenue trends, top customers, best-performing products, and customer segments.

## Tools Used

* MySQL
* MySQL Workbench
* SQL
* CSV dataset

## Dataset

The dataset contains 300 sales records with order details, customer details, region, product category, sales, and profit.

## Business Questions Answered

1. What are total sales and profit?
2. Which region generates the highest sales?
3. What is the monthly sales trend?
4. Who are the top 10 customers?
5. Which products generate the most revenue?
6. How can customers be segmented by spending?
7. What is each category's revenue contribution?
8. Which month had the highest sales?

## SQL Concepts Used

* SELECT
* GROUP BY
* Aggregate functions
* CASE WHEN
* Subqueries
* Window functions
* DATE\_FORMAT

## Key Insights

- Rohan Mehta is the highest revenue-generating customer (~305k)
- Top 5 customers contribute a significant share of total revenue
- Revenue is concentrated among a few high-value customers
- Opportunity exists to increase revenue by targeting mid-tier customers


## Resume Bullet

Sales \& Customer Analysis using SQL: Analyzed 300 sales transactions using MySQL to identify sales trends, top customers, product performance, and customer segments using GROUP BY, CASE statements, subqueries, and window functions.
